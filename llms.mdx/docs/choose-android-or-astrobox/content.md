# 选择合适的接收端 (/docs/choose-android-or-astrobox)



Android 与 AstroBoxV2 都能连接 Shell++ 手环端，但它们面向的工作方式不同。你只需要选择其中一个接收端。

## 选择你的使用方式 [#选择你的使用方式]

| 你的设备                  | 推荐接收端           | 适合的场景                     |
| --------------------- | --------------- | ------------------------- |
| Android 手机或平板         | Shell++ Android | 日常连接、截图同步、预览导出与远程终端       |
| Windows、macOS 或 Linux | AstroBoxV2 插件   | 桌面管理、截图同步、终端与自动化调用        |
| iPhone 或 iPad         | AstroBoxV2 插件   | 通过 AstroBoxV2 所在设备完成连接与同步 |

无论选择哪条链路，手环端您都需要下载并安装lua表盘和快应用。

## 选择 AndroidAPP，如果你…… [#选择-androidapp如果你]

* 主要在安卓手机上查看和保存截图。
* 希望使用完整的日常配套界面。
* 需要把截图保存到系统相册，或生成带设备边框的图片。
* 希望在同一应用内查看连接、传输进度和最近任务。

## 选择 AstroBoxV2插件，如果你…… [#选择-astroboxv2插件如果你]

* 主要在电脑或非安卓平台上调试或维护设备。
* 已经使用 AstroBoxV2 管理 Vela 设备。
* 需要在插件中查看截图、终端、调试信息和日志。
* 希望通过 Deeplink 或命令行把部分操作接入自动化流程。

接下来先确认[支持的设备与平台](/docs/supported-devices-and-platforms)。
